# POS Pro — Static Website

This is a minimal static website for the POS Pro marketing site. It contains pure HTML and CSS (no JavaScript).

Files:
- `index.html` — Home
- `about.html` — About
- `contact.html` — Contact form
- `privacy.html` — Privacy policy
- `styles.css` — Main styles

To preview locally, open `index.html` in your browser or serve the folder with a static server (e.g., `npx http-server` or `python -m http.server`).
